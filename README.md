# Task Management Web App 

A full-stack Task Management application built with the MERN stack. This app helps users create, manage, update, and track tasks with a clean and responsive UI.

---

## Features

- User Authentication (Login/Register)
- Create, Read, Update, Delete (CRUD) tasks
- Mark tasks as completed / pending
- Filter tasks (All / Completed / Pending)
- User profile management
- Real-time UI updates (no manual refresh needed)
- Fully responsive design
- Clean dashboard with task stats

---

## Tech Stack

### Frontend
- React.js
- React Context API
- Axios
- Tailwind CSS
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt.js
